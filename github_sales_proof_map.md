# GitHub Sales Proof Map

| Industry / offer | Best proof | What it demonstrates | Limitation to state |
|---|---|---|---|
| Printing and packaging | Printing Press ERP | Workflow stages, dashboards, role-aware operations | Demo/client access may be restricted |
| Quotation calculators | Timber / curtain / wallpaper calculator | Rule-based calculation and practical UI | Domain formulas must be confirmed with each business |

| Lead and follow-up systems | Career Pipeline Dashboard | CSV data loading, filters, priorities, follow-ups | V1 browser changes use localStorage, not shared cloud state |
| Startup trial work | Career Application Kit repository | Documentation, CI, testing, deployment, public proof | AI-assisted workflow must be described honestly |

Use proof to start a discovery conversation, not to claim measured business results that were not recorded.
