# Daily Sales Workflow

1. Open Dashboard -> Sales -> Today.
2. Verify up to five public contacts and observations.
3. Review the matching offer and email.
4. Approve or reject each draft manually.
5. Send approved messages yourself.
6. Mark sent and add the date.
7. Process replies and set follow-ups.
8. Export JSON at the end of the session.
9. Save one reusable lesson weekly.
10. Stop after 15-30 minutes.
